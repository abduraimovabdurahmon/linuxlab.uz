## ☕ Java Dasturlash Tili Haqida

Java — bu **ob'ektga yo'naltirilgan dasturlash tili** bo‘lib, keng tarqalgan va turli sohalarda qo‘llaniladi.  
U **platformadan mustaqil**, yuqori samaradorlik va xavfsizlikni ta’minlaydi.

Java ilovalari **kompyuter arxitekturasidan qat’i nazar** har qanday platformada ishlay oladi.  
Sintaksisi esa **C va C++** tillariga o‘xshaydi, bu esa boshqa tillardan o‘tishni osonlashtiradi.

## ⚙️ Java ni o'rnatish

<Tabs items={['Windows', 'Linux', 'MacOS']}>

<Tabs.Tab>
**Windows** — Java’ni o‘rnatish uchun:

1. [Java rasmiy veb-saytiga](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) o'ting.  
2. "JDK Download" bo‘limidan Windows uchun JDK ni yuklab oling.  
3. Yuklab olingan faylni ishga tushiring va o‘rnatish jarayonini bajaring.  
4. "System Properties" → "Environment Variables" ga kiring.  
5. "Path" o‘zgaruvchisiga JDK `bin` papkasi yo‘lini qo‘shing (masalan: `C:\Program Files\Java\jdk-11\bin`).  
6. O‘zgarishlarni saqlang va kompyuterni qayta yuklang.  

```bash copy
java -version
```

</Tabs.Tab>

<Tabs.Tab>
**Linux** — Java’ni o‘rnatish uchun:

1. Terminalni oching.
2. JDK o‘rnatish:

```bash copy
sudo apt update
sudo apt install default-jdk
```

3. Tekshirish:

```bash copy
java -version
```

</Tabs.Tab>

<Tabs.Tab>
**MacOS** — Java’ni o‘rnatish uchun:

1. [Java rasmiy veb-saytiga](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) o'ting.
2. "JDK Download" bo‘limidan MacOS uchun JDK ni yuklab oling.
3. Yuklab olingan faylni ishga tushiring va o‘rnatish jarayonini bajaring.

## ✏ Java sintaksisi

Java dasturlash tilining asosiy sintaksisi bilan tanishamiz:

```java copy
public class Main {
    public static void main(String[] args) {
        System.out.println("Salom, Dunyo!");
    }
}
```

Java da class nomlari doim bosh harf bilan boshlanishi kerak, shuning uchun biz kodimizga ``Main.java`` deb nom berdik.

## 🏃 Java kodini ishlatish

Java kodini ishlatish uchun quyidagi qadamlarni bajaring:

1. **Kodni kompilyatsiya qilish**: Terminal yoki cmdni oching va kod joylashgan papkaga o'ting. Keyin quyidagi buyruqni bajaring:

```bash copy
javac Main.java
```

Bu buyruq `Main.java` faylini kompilyatsiya qiladi va `Main.class` faylini yaratadi.

2. **Kodni ishga tushirish**: Kompilyatsiya qilingan kodni ishga tushirish uchun quyidagi buyruqni bajaring:

```bash copy
java Main
```

Natijada, terminalda "Salom, Dunyo!" xabari ko'rsatiladi.
